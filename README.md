# PingDesk — Site Institucional

Site institucional da **PingDesk**, help desk especializado para provedores de internet.

## Tecnologias

- HTML5 puro (sem frameworks ou dependências)
- CSS3 com variáveis customizadas
- Google Fonts: Space Mono + DM Sans

## Estrutura

```
pingdesk-site/
├── index.html                  # Página principal
├── android-chrome-192x192.png  # Logo PingDesk
└── README.md
```

## Como usar

Basta abrir o `index.html` no navegador ou subir os arquivos para qualquer servidor estático (Nginx, Apache, Netlify, GitHub Pages, etc.).
